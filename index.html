Data Visualization:

d3.chart('BarChart')

tags:angularjs,d3,d3.chart,nodejs,mongodb

This example illustrates an ensemble of several technologies with nodejs as the server, mongodb as the datastore, express as the webframework, angularjs as the frontend automation and d3 with d3.chart to create reusable barcharts from a given set of data served from mongodb.

You can find the sourcecode on Github at andreas-marschke/example-mongo-barchart.
